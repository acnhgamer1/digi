# Digi - Your Digital Assistant

Digi is an AI-powered digital assistant inspired by Pal from "The Mitchells vs. The Machines". This web application features voice recognition, AI responses, and animated facial expressions that sync with speech.

## Features

- 🎤 **Voice Recognition**: Speak to Digi using your microphone
- 🗣️ **Text-to-Speech**: Digi responds with natural-sounding speech
- 🤖 **AI Responses**: Powered by Pollinations AI for intelligent conversations
- 😊 **Animated Face**: Dynamic facial expressions that react to conversation state
- 📱 **Responsive Design**: Works on desktop and mobile devices
- 🌐 **GitHub Pages Ready**: Deployed and accessible via GitHub Pages

## How to Use

1. Visit the live demo at: `https://acnhgamer1.github.io/digi/`
2. Click the "Talk to Digi" button
3. Speak your question or command
4. Watch Digi's face animate as it listens, thinks, and responds

## Technical Details

### Voice Features
- Uses Web Speech API for voice recognition
- Attempts to use female voices similar to Alexa
- Lip-sync animations during speech
- Real-time transcript display

### AI Integration
- Integrates with Pollinations AI for intelligent responses
- Designed to respond like a helpful digital assistant
- Handles errors gracefully with fallback responses

### Animations
- CSS-based facial animations
- Different states: listening, thinking, speaking
- Smooth transitions and visual feedback
- Pal-inspired design aesthetic

## Browser Compatibility

- Chrome/Chromium (recommended)
- Firefox (limited speech synthesis voices)
- Safari (iOS/macOS)
- Edge

**Note**: Speech recognition works best in Chrome-based browsers.

## Development

To run locally:
1. Clone this repository
2. Open `index.html` in a web browser
3. Allow microphone permissions when prompted

## Deployment

This app is configured to run on GitHub Pages. Simply push to the main branch and enable GitHub Pages in repository settings.

## Privacy

- All voice processing happens in your browser
- No audio data is stored or transmitted
- AI requests are sent to Pollinations AI (see their privacy policy)
